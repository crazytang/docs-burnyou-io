# 概述


### BurnYou是什么？
BurnYou是一个通过回收玩家NFT后代运营NFT对战赛事，促成被低估的NFT交易与IP孵化的区块链平台。

玩家将被低估的NFT丢进BurnYou 回收站，Burnyou进行赛事运营其中主要包括NFT 1v1单挑战，以及NFT合集团灭战。所有赛事均通过大众投票的形式，决出对战双方的最终胜负，并将败方NFT销毁，胜方获得推广资金与社区关注度等支持。完成【次物回收，重塑价值】。胜利玩家收到增值收益与代币奖励，失败玩家亦可收到一定数量的代币奖励，NFT合集方通过回收站与赛事增加了曝光度。

### BurnYou背后的动机是什么？
通过对NFT市场进行大面积的优胜劣汰，促进优秀NFT作品的价值上升，在区块链新世界中孵化出属于新一代年轻人自己的新IP。


### BurnYou的强项是什么？
- NFT参赛玩家获胜后通过败方的推广费可以升值您的NFT甚至整个NFT合集，反之败方则销毁价格归零
- NFT参赛玩家参与费用为0 USDT——作为参赛玩家您无需担心报名费或其他任何的费用，甚至还能享受早期参与代币空投的奖励
- NFT 投票玩家通过放大镜可以参与Vote to Earn挖矿活动
- NFT赛事观众能通过BurnYou挑选到“NFT金狗”——价格低但潜力大的NFT
- NFT 举牌拍卖玩家可以通过押宝形式，预测哪一边的NFT将会获胜，这是BurnYou独有的拍卖交易模式，ONLY in BurnYou


### BurnYou是否可以升级或者更改？
可以，虽然BurnYou没有管理员密钥，但社区可以通过持有已流通代币比例发起治理投票以更改规则，否则智能合约代码是完全不变的。

### BurnYou的Battle形式是什么？
- NFT 1v1 Battle
    - 任意NFT均可摆擂或挑擂，互相选择对手开战；由大众投票决定输赢，落败方NFT将被销毁。
- NFT合集之间的团灭战
  - 在1v1 Battle的基础上，败方落败后24小时内，同一合集的其他NFT可以参战拯救它，直到某一方无人拯救，则落败并集体销毁。
  - 围观者押宝
  
  双方Battle过程中，围观者可以对任一方的NFT进行固定4%加价抢购。但假如抢到的NFT落败，则所有押宝的加价金额，将被投入胜方推广池。
  - 胜方推广池
  
  任何Twitter玩家，都可以将自己的头像换成胜方NFT图片，挂满24小时，即可以粉丝数为权重分取该胜方推广池中的金额。

### BurnYou是否收取任何费用？
- 您不需要支付任何进场或出场费用
- 每当您的一个NFT获胜并交易后，BurnYou与您共享升值的权益将收取一定比例的交易手续费

### 我会亏本吗？
- 作为非保管性的系统，发送给协议的所有令牌都将通过算法进行保存和管理，不会受到任何人或机构的干扰。这意味着您的资金将仅受智能合约代码中规定的规则约束
- 参加NFT对战，输方会丧失整个NFT的价值，而胜方有可能可以成倍升值（取决于投票数与推广资金池）

请注意，尽管我们队系统进行了非常认真的审核，但我们无法完全排除黑客或系统漏洞导致的玩家损失（请参阅免责声明）
