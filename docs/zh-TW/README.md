# 概述


### BurnYou是什麽？
BurnYou是壹個通過回收玩家NFT後代運營NFT對戰賽事，促成被低估的NFT交易與IP孵化的區塊鏈平台。

玩家將被低估的NFT丟進BurnYou 回收站，Burnyou進行賽事運營其中主要包括NFT 1v1單挑戰，以及NFT合集團滅戰。所有賽事均通過大衆投票的形式，決出對戰雙方的最終勝負，並將敗方NFT銷毀，勝方獲得推廣資金與社區關注度等支持。完成【次物回收，重塑價值】。勝利玩家收到增值收益與代幣獎勵，失敗玩家亦可收到壹定數量的代幣獎勵，NFT合集方通過回收站與賽事增加了曝光度。

### BurnYou背後的動機是什麽？
通過對NFT市場進行大面積的優勝劣汰，促進優秀NFT作品的價值上升，在區塊鏈新世界中孵化出屬于新壹代年輕人自己的新IP。


### BurnYou的強項是什麽？
- NFT參賽玩家獲勝後通過敗方的推廣費可以升值您的NFT甚至整個NFT合集，反之敗方則銷毀價格歸零
- NFT參賽玩家參與費用爲0 USDT——作爲參賽玩家您無需擔心報名費或其他任何的費用，甚至還能享受早期參與代幣空投的獎勵
- NFT 投票玩家通過放大鏡可以參與Vote to Earn挖礦活動
- NFT賽事觀衆能通過BurnYou挑選到“NFT金狗”——價格低但潛力大的NFT
- NFT 舉牌拍賣玩家可以通過押寶形式，預測哪壹邊的NFT將會獲勝，這是BurnYou獨有的拍賣交易模式，ONLY in BurnYou


### BurnYou是否可以升級或者更改？
可以，雖然BurnYou沒有管理員密鑰，但社區可以通過持有已流通代幣比例發起治理投票以更改規則，否則智能合約代碼是完全不變的。

### 如何使用BurnYou？
首先，您需要點擊BurnYou.io，鏈接您的錢包，授權您的NFT，開始等待或接受對戰，同時您可以參與其他賽事的投票，挖掘NFT金狗

### BurnYou的Battle形式是什麽？
- NFT 1v1 Battle
  - 任意NFT均可擺擂或挑擂，互相選擇對手開戰；由大衆投票決定輸贏，落敗方NFT將被銷毀。
- NFT合集之間的團滅戰
  - 在1v1 Battle的基礎上，敗方落敗後24小時內，同壹合集的其他NFT可以參戰拯救它，直到某壹方無人拯救，則落敗並集體銷毀。
  - 圍觀者押寶

  雙方Battle過程中，圍觀者可以對任壹方的NFT進行固定4%加價搶購。但假如搶到的NFT落敗，則所有押寶的加價金額，將被投入勝方推廣池。
  - 勝方推廣池

  任何Twitter玩家，都可以將自己的頭像換成勝方NFT圖片，挂滿24小時，即可以粉絲數爲權重分取該勝方推廣池中的金額。

### BurnYou是否收取任何費用？
- 您不需要支付任何進場或出場費用
- 每當您的壹個NFT獲勝並交易後，BurnYou與您共享升值的權益將收取壹定比例的交易手續費

### 我會虧本嗎？
- 作爲非保管性的系統，發送給協議的所有令牌都將通過算法進行保存和管理，不會受到任何人或機構的幹擾。這意味著您的資金將僅受智能合約代碼中規定的規則約束
- 參加NFT對戰，輸方會喪失整個NFT的價值，而勝方有可能可以成倍升值（取決于投票數與推廣資金池）
