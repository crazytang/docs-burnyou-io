# 投票（放大镜）

### 什么是放大镜？
放大镜是一款代表玩家对BurnYou平台上的NFT Battle所拥有的投票权与承载该玩家对藏品鉴赏力水平的NFT

### 为什么需要放大镜来投票？
- 因为创建N个加密钱包的成本为0而投票却是按数量结算胜败，所以为了防止刷票则需要一项准入证明来保证每一次投票的公正性
- 放大镜是投票玩家鉴赏水平的体现
- 放大镜是投票玩家挖矿的凭证（Vote2Earn）

### 如何使用放大镜来投票？
- 一个钱包账户只能拥有一个放大镜
- 一个放大镜对同一场Battle只能投一次票

### 如何获得放大镜？
- 购买铸造
- 白名单铸造（免费）

### 放大镜可以转移吗？
不可以

### 每个放大镜都是一样的吗？
铸造出来的放大镜都是一样的，但随着参与投票的过程放大镜也有变化的可能。放大镜的变化简单可以分为升级与降级，从而影响投票的权重与挖矿的多少。

### 哪些行为可以决定放大镜的变化？
- 投票次数
- 投中次数
- 投错次数
