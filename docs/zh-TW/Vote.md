# 投票（放大鏡）

### 什麽是放大鏡？
放大鏡是壹款代表玩家對BurnYou平台上的NFT Battle所擁有的投票權與承載該玩家對藏品鑒賞力水平的NFT

### 爲什麽需要放大鏡來投票？
- 因爲創建N個加密錢包的成本爲0而投票卻是按數量結算勝敗，所以爲了防止刷票則需要壹項准入證明來保證每壹次投票的公正性
- 放大鏡是投票玩家鑒賞水平的體現
- 放大鏡是投票玩家挖礦的憑證（Vote2Earn）

### 如何使用放大鏡來投票？
- 壹個錢包賬戶只能擁有壹個放大鏡
- 壹個放大鏡對同壹場Battle只能投壹次票

### 如何獲得放大鏡？
- 購買鑄造
- 白名單鑄造（免費）

### 放大鏡可以轉移嗎？
不可以

### 每個放大鏡都是壹樣的嗎？
鑄造出來的放大鏡都是壹樣的，但隨著參與投票的過程放大鏡也有變化的可能。放大鏡的變化簡單可以分爲升級與降級，從而影響投票的權重與挖礦的多少。

### 哪些行爲可以決定放大鏡的變化？
- 投票次數
- 投中次數
- 投錯次數
