# $BURN獎勵與分配

### 什麽是$BURN？
$BURN是基于以太坊主網發行的ERC20代幣，同時是BurnYou智能合約的治理代幣，對BurnYou智能合約代碼擁有治理權，並擁有該智能合約國庫資金的投票權和撥款權

### $BURN的最大供應量是多少？
$BURN的最大供應量爲1，000,000,000個代幣

### $BURN的配發時間表是什麽？
社區發行（在LP激勵和社區儲備之外）遵循每年減半的時間表，由以下函數描述：510,000,000*(1–0.5^year)。設置這條發行曲線的目的是有力地激勵早期采用者，同時保持壹定的長期激勵
