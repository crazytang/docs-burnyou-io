# $BURN奖励与分配

### 什么是$BURN？
$BURN是基于以太坊主网发行的ERC20代币，同时是BurnYou智能合约的治理代币，对BurnYou智能合约代码拥有治理权，并拥有该智能合约国库资金的投票权和拨款权

### $BURN的最大供应量是多少？
$BURN的最大供应量为1，000,000,000个代币

### $BURN的配发时间表是什么？
社区发行（在LP激励和社区储备之外）遵循每年减半的时间表，由以下函数描述：510,000,000*(1–0.5^year)。设置这条发行曲线的目的是有力地激励早期采用者，同时保持一定的长期激励
