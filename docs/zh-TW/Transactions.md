# 交易（押宝拍卖） <font size='4em'><i>ONLY in BurnYou</i></font>

### 为什么我的NFT Battle获胜后可以升值？
因为在Battle开始的同时，BurnYou上进行着独有的一种押宝拍卖模式来获得胜方NFT。就像现实世界的拍卖场景一样，押宝式拍卖上交易玩家通过举牌出价的形式，每一次举牌加价4%，来竞价获得胜出的NFT

### 如果我押宝的NFT Battle战败了怎么办？
那么很遗憾，您押宝的加价金额将被全部放入胜方NFT的推广池中，用于胜方NFT在推特上的推广费用；
反之亦然，您押宝拍卖到的NFT将在推特上被疯狂宣传

### 什么是Battle推广池？
Battle推广池是BurnYou为了增加NFT流动性的独有模式，Battle开始时，玩家可以竞猜押宝哪一边的NFT将会获胜。一旦玩家押宝了其中一方就意味将手中的ETH举牌参加NFT拍卖会，换句话来说，玩家可以享受到获胜NFT的升值或贡献自己的资金给予获胜的NFT

### 举牌押宝玩家有挖矿奖励吗？
有的。且，您也可以同时是举牌押宝玩家与Battle参赛玩家，更有甚至，您还可以同时是投票玩家。请记住BurnYou上所有的玩家角色有挖矿奖励，所以尽可能多试试BurnYou的各种角色是您最佳的选择
