# 交易（押寶拍賣） <font size='4em'><i>ONLY in BurnYou</i></font>

### 爲什麽我的NFT Battle獲勝後可以升值？
因爲在Battle開始的同時，BurnYou上進行著獨有的壹種押寶拍賣模式來獲得勝方NFT。就像現實世界的拍賣場景壹樣，押寶式拍賣上交易玩家通過舉牌出價的形式，每壹次舉牌加價4%，來競價獲得勝出的NFT

### 如果我押寶的NFT Battle戰敗了怎麽辦？
那麽很遺憾，您押寶的加價金額將被全部放入勝方NFT的推廣池中，用于勝方NFT在推特上的推廣費用；
反之亦然，您押寶拍賣到的NFT將在推特上被瘋狂宣傳

### 什麽是Battle推廣池？
Battle推廣池是BurnYou爲了增加NFT流動性的獨有模式，Battle開始時，玩家可以競猜押寶哪壹邊的NFT將會獲勝。壹旦玩家押寶了其中壹方就意味將手中的ETH舉牌參加NFT拍賣會，換句話來說，玩家可以享受到獲勝NFT的升值或貢獻自己的資金給予獲勝的NFT

### 舉牌押寶玩家有挖礦獎勵嗎？
有的。且，您也可以同時是舉牌押寶玩家與Battle參賽玩家，更有甚至，您還可以同時是投票玩家。請記住BurnYou上所有的玩家角色有挖礦獎勵，所以盡可能多試試BurnYou的各種角色是您最佳的選擇
