# 對戰

### 爲什麽我要使用BurnYou進行NFT對戰？
- 您對戰獲勝的NFT將有可能升值
  <br>制造沖突，幫助優勝方吸引眼球；汰弱留強，減少市場供應量，使有限資源集中給優質方
- 您的對戰過程是您NFT及合集的展示曝光過程
  <br>即使您手中的NFT戰敗被銷毀，所屬NFT合集也得到了展示曝光
  <br>銷毀機制減少同壹合集的供應量；
- 您參與對戰所獲得的BurnYou平台代幣有可能升值（NFT行業進入下半場，上半場屬于生産者，下半場屬于流通平台類似BurnYou）

### 所謂的銷毀是什麽意思？
銷毀NFT並不是物理世界的破壞打碎，而是智能合約自動發送到壹個無私鑰的“0地址”，而沒人能拿得出來達到的銷毀效果

### 以太坊是BurnYou唯壹接受的對戰公鏈嗎？
BurnYou是圍繞以太坊生態建立的應用。
<br>如果您的NFT在以太坊主網上，您在以太坊L1上將NFT授權與質押後，未來所有的Battle、交易結算都將在以太坊L2 ZK-Sync上發生
<br>如果您的NFT已經在ZK-Sync上，將可以直接開始Battle和交易，享受以太坊二層網絡低廉的使用成本。

### 對戰的流程是怎麽樣的？
- 發起
  <br>任何玩家可發起擂台，等待對手應戰
  或是直接參與已有的擂台
- 對戰中
  <br>采用放大鏡投票方式，持續時間24小時
- 對戰後
  <br>公布結果票數多的壹方勝出獲得推廣池，票數少的銷毀
