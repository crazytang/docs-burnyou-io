# 对战

### 为什么我要使用BurnYou进行NFT对战？
- 您对战获胜的NFT将有可能升值
  <br>制造冲突，帮助优胜方吸引眼球；汰弱留强，减少市场供应量，使有限资源集中给优质方
- 您的对战过程是您NFT及合集的展示曝光过程
  <br>即使您手中的NFT战败被销毁，所属NFT合集也得到了展示曝光
  <br>销毁机制减少同一合集的供应量；
- 您参与对战所获得的BurnYou平台代币有可能升值（NFT行业进入下半场，上半场属于生产者，下半场属于流通平台类似BurnYou）

### 所谓的销毁是什么意思？
销毁NFT并不是物理世界的破坏打碎，而是智能合约自动发送到一个无私钥的“0地址”，而没人能拿得出来达到的销毁效果

### 以太坊是BurnYou唯一接受的对战公链吗？
BurnYou是围绕以太坊生态建立的应用。
<br>如果您的NFT在以太坊主网上，您在以太坊L1上将NFT授权与质押后，未来所有的Battle、交易结算都将在以太坊L2 ZK-Sync上发生
<br>如果您的NFT已经在ZK-Sync上，将可以直接开始Battle和交易，享受以太坊二层网络低廉的使用成本。

### 对战的流程是怎么样的？
- 发起
<br>任何玩家可发起擂台，等待对手应战
或是直接参与已有的擂台
- 对战中
  <br>采用放大镜投票方式，持续时间24小时
- 对战后
  <br>公布结果票数多的一方胜出获得推广池，票数少的销毁
